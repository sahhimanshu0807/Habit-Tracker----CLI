# Habit Tracker -- CLI

A python [CLI](https://github.com/resources/articles/what-is-a-cli) (command line interface) application to track habit.

## Introduction
This application adds, removes, marks as completed, and delete daily habits as requested by the user. It shows what percentage of the days those habits where followed in the current month as well as in the current year.
The plan is to let the user create an account and simultaneously create a file for that user in the disk. After that all the tasks and activities of the user are stored in that file, thus reducing the need to search through data. The filename will always be the username which can't be changed once the profile has been created.
